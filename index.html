Analista PCM Jr

Tempo Integral
Nov/2020 - Out/2022 (2 anos)
Com o objetivo de aumentar a performance de manutenção industrial, tive o desafio de implementar controles e desenvolver respostas rápidas para direcionamento do time com base em dados. O grande obstáculo foi a captura de dados do campo para alimentar o sistema ERP (SAP – PM) e assim conseguir ter atualizações nos relatórios e dashboards com mais rapidez para gerar analises e ações para o time de manutenção industrial agir de forma eficiente.
Trabalhei desenvolvendo aplicativos web para coleta de apontamentos e alimentação do sistema SAP – PM com a construção de scripts em VBA. Utilizei ferramentas como Power Apps para o desenvolvimento de aplicativo low-code, Sharepoint para armazenar dados, Power Bi para a visualização das informações e o Power Automate para gerar arquivos em Excel com o intuito de rodar o script VBa para alimentar o sistema SAP. Também tive o desafio de controlar os custos de manutenção, esse tarefa me possibilitou a oportunidade de construir um RPA (Robotic Process Automation) para a extração de dados do sistema SAP e alimentando uma base do Power BI. Assim consegui desenvolver um dashboard de custos para o setor de manutenção com atualizações diárias. O projeto de RPA foi um sucesso e tive a oportunidade de expandir para toda a indústria.

Projetos em Ciência de Dados

Digitalização em Rotas de Inspeção Sensitiva com Software Low-Code

Este não é um projeto focado em ciência de dados, porém foi um projeto que conduzi na Tereos Amidos & Adoçantes e também foi meu Trabalho de Conclusão de Curso no MBA de Engenharia de Manutenção na UNIMAR. Durante meu trabalho como analista de PCM, participei da implantação da metodologia TPM, mais especificamente o pilar de manutenção autônoma. E uma das rotinas geradas por esse pilar é a rota de inspeção sensitiva que deve ser realizada pela operação. No entanto o fluxo de trabalho dessa rotina não estava otimizado. A inspeção tinha o roteiro impresso e os dados coletados eram anotados na folha de papel, e assim que finalizado era levado até um colaborador de manutenção para que as informações coletadas fossem digitadas em uma planilha Excel. Sendo assim um processo moroso.
A solução foi trazida por mim, com o projeto de criar um aplicativo low-code que impactaria no fluxo de trabalho, deixando mais eficiente, trazendo maior tempo de resposta do time de manutenção e também maior confiabilidade nos dados.
Ferramentas utilizadas: Power Apps, Power Automate, Power Bi e Sharepoint.
